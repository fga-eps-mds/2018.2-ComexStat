Comex Stat API

1. Sobre a API

Essa é uma API dedicada a prover acesso aos dados relacionados ao comércio exterior de bens, provenientes do Ministério da Indústria, Comércio Exterior e Serviços (MDIC). A API é feita em GraphQL, cuja ferramenta de busca a banco de dados oferece recursos que facilitam as buscas e filtros nas pesquisas

2. Como usar

A estrutura de consultas na API é constituida de uma string em formatada em JSON contendo a operação a ser feita (que no caso dessa API sempre será um query), o método de busca a ser utilizado, com seus respectivos filtros, e os campos pesquisados. Como no exemplo a seguir:

No exemplo acima é primeiro definida a operação "query", dentro da qual é utilizado o método de busca "allImport" com um filtro que filtra pelo nome do país de origem (os filtros ficam entre parênteses), e o único campo requerido é o "ncm", que por sua vez tem um campo requerido chamado "countryNamePt". Há ainda, separando o método de busca e os campos, uma estrutura com "edges" e "node". Essas são apenas obrigatoriedades estruturais da linguagem que definem uma melhor organização dos resultados que serão mostrados. Em seguida, basta executar o query através do botão "Play" na API e os resultados serão retornados em formato JSON no container da direita

3. Métodos de buscas

A seguir estão descritos todos os possíveis métodos de busca que podem ser utilizados no query e seus respectivos valores de retorno. Bem como os filtros cabíveis a cada método.

3.1. allImport

Buscar por todas as importações

3.2. allExport

Buscar por todas as expotações

3.3. allINcm

Buscar por Ncm - Nomenclatura Comum do Mercosul

3.4. allTradeblocs

Buscar por blocos econômicos

3.5. allCountry

Buscar por paises

3.6. allFederativeunit

Buscar por unidade federativa

3.7. allTransportation

Buscar por via(modal) de transporte

3.8. allUrf

Buscar por Urf - Unidade da Receita Federal

3.9. allCuci

Buscar por CUCI - Classificação Uniforme para Comércio Internacional

3.10. allCgce

Buscar por CGCE - Classificação por Grandes Categorias Econômicas

3.11. allSh

Buscar SH - Sistema Harmonizado
